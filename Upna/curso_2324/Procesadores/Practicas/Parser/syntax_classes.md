# Clases sintacticas para las palabras clave
## Palabras reservadas
Tanto en minúscula como mayúscula como mezcla en cualquier forma. No pueden ser usadas como identificadores (pueden haber mas):
- de
- dev
- e/s
- ref
- const
- fconst
